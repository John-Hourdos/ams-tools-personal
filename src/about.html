Trajectory Tools

Version 0.1 (beta)

About

Trajectory Tools is a PyQt6-based application for analyzing and visualizing vehicle trajectory data stored in SQLite databases.

Features

• Import and analyze trajectory data
• Time-space plotting and visualization
• Export capabilities for data analysis
• SQLite database management

Requirements

• Python 3.9+
• PyQt6
• SQLite3

For more information, please refer to the documentation.
